# Dijital Journal

Dijital Journal, günlük notlarını kolayca oluşturup düzenleyebileceğin basit ve modern bir React uygulamasıdır.

Uygulama içinde tarih, başlık, günlük metni ve ruh hali bilgisiyle kayıt ekleyebilir; kayıtları düzenleyebilir ve silebilirsin.

## Özellikler

- Yeni günlük kaydı ekleme
- Mevcut kaydı düzenleme
- Kayıt silme
- Ruh hali seçimi (mutlu, normal, üzgün, sinirli)
- Verileri tarayıcıda localStorage ile saklama
- Bootstrap ile responsive arayüz

## Kullanılan Teknolojiler

- React
- React Hooks (useState, useEffect)
- Bootstrap 5
- localStorage

## Kurulum

1. Depoyu klonla:

	git clone <repo-link>

2. Proje klasörüne gir:

	cd digital-journal

3. Bağımlılıkları yükle:

	npm install

4. Geliştirme ortamında çalıştır:

	npm start

Uygulama varsayılan olarak şu adreste açılır:
http://localhost:3000

## Kullanılabilir Komutlar

- npm start: Geliştirme sunucusunu başlatır
- npm run build: Production build oluşturur
- npm test: Test ortamını başlatır

## Netlify Deploy

Projede Netlify için yapılandırma dosyası bulunur.

GitHub repo Netlify ile bağlandığında, production branch'e her push işleminden sonra otomatik deploy tetiklenir.
